MongoDB: NoSQL documentDB
  - NoSQL -> data stored and organized but not in rows/cols
  - documentDB -> data stored in documents which are organized using collections

Documents
  - {
      "field": "value"
      "uniqueIdentifier": "data"
      num: 7
    }

MongoDB Atlas: Cloud database for runnign MongoDB
Replica set: connected machines with the same data
Instance: single machine running software (ie mongodb db)
Clusters: group of servers that store data

Connecting to Atlas Cluster:
